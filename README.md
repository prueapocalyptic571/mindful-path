## 🌿 Overview

mindful-path is a daily habit tracker for students. It helps you track simple routines, stay on task, and keep a clear view of your progress. The app uses a calm, clean layout so you can use it each day without clutter.

It is built for Windows users who want a direct way to download and run the app from GitHub releases.

## ✅ What you can do

- Track daily habits in one place
- Mark habits as done for each day
- View your progress over time
- Keep a simple list of school and personal goals
- Use a layout that stays easy to read
- Build steady routines with small steps

## 📘 How to use mindful-path

Use the app each day with a simple routine:

1. Open the app
2. Review your habits for the day
3. Mark each habit when you complete it
4. Check your progress at the end of the week
5. Adjust your list if a habit feels too hard

Keep your habit list small at first. Three to five habits is a good start for most students.

## 🛠️ Common use tips

- Use short habit names
- Keep one list for school tasks and one for personal growth
- Check the app at the same time each day
- Start with easy habits before adding more
- Remove habits that no longer fit your schedule
- Use the progress view to stay on track
